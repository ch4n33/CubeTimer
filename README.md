#   CubeTimer
##  Dependencies
*   python>3.10/
*   PyQt5
*   numpy
##  How to use?
*   `$ git clone ~`
*   install dependencies
    *   `$ pip install PyQt5 numpy`
*   `$ python main.py`

*   you can build this code on your own, by pyinstaller, etc.
    *   `$ pip install -U pyinstaller`
    *   `$ pyinstaller -w --onefile main.py`
